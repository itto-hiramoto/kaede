# comment_app

1. Put your slide HTML at `frontend/slides/slide.html`.
   The repository ships a dummy slide by default, so replace it with your own file.

2. Start the server.

```bash
cd example/comment_app
kaede run
```

If you do not have an installed `kaede` binary:

```bash
cd example/comment_app
cargo run -- run
```

3. Open these pages in your browser.

- `http://localhost:8080/`
- `http://localhost:8080/screen`
